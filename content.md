# 🎼 Music School Website – AI Prompt for Vibe Coding (Enhanced)

## 🎯 Goal

Generate a **modern, artistic, elegant music school website UI** that includes:

* Video background hero
* Teacher introduction
* Image & video gallery
* Trial booking form (lead capture for CRM)
* All content must be written in **Vietnamese with full diacritics**
* The website is for a **Vietnamese audience**, so the tone must be natural, local, and not mixed with English

---

## 🧠 Master Prompt (Updated)

Use this prompt with AI tools (v0, Framer AI, Cursor, GPT, etc.):

```txt id="g8x9pl"
Create a modern, artistic, and elegant website UI for a music school.

Style:
- Dark luxury theme (black / deep gray background)
- Gold / beige accent colors
- Minimalist layout with lots of whitespace
- Typography: mix of serif (elegant) and sans-serif (clean)
- Smooth animations (fade, hover, scroll)
- Premium, cinematic feeling

Key Requirement:
- The homepage must include a FULLSCREEN VIDEO BACKGROUND hero (autoplay, muted, loop)
- The video should showcase piano, violin, or artistic classroom scenes

Pages:

1. Home Page:
- Hero:
  - Fullscreen video background (music performance / classroom / piano close-up)
  - Overlay text:
    "Where Art Meets Discipline"
  - CTA:
    "Book a Trial Lesson"
    "Explore Courses"

- Sections:
  - About (short intro of the music center)
  - Courses preview (Piano, Violin, Guitar, Vocal)
  - Teachers preview (highlight instructors)
  - Gallery preview (images + videos)
  - Why choose us
  - Final CTA

2. Course Page:
- Banner with course name (Piano / Violin...)
- Description
- Learning path (Beginner → Advanced)
- Benefits
- Images / short video
- CTA: Book trial lesson

3. Booking Page (Lead Capture):
- Clean modern form:
  - Name
  - Phone
  - Email
  - Course selection
  - Time slot selection
  - Notes
- CTA: Submit
- Show success state after submit

4. Teachers Page:
- Grid layout:
  - Teacher photo
  - Name
  - Instrument
  - Experience
  - Short bio
- Elegant card design

5. Gallery Page:
- Mix of:
  - Image grid
  - Video thumbnails (click to play modal)
- Showcase performances, classes, events

6. About Page:
- Story of the music center
- Philosophy and teaching approach
- Studio environment (images/video)

Components:
- Sticky navbar (transparent → solid on scroll)
- Footer with contact info + social links
- Reusable cards (course, teacher)
- CTA sections
- Video modal player
- Form components

UX Flow:
- User watches video hero → gets emotional connection
- Scrolls → sees courses + teachers + gallery
- Clicks "Book Trial Lesson"
- Fills form → sees success message

Important:
- Focus on UI only (no backend required yet)
- Use realistic placeholder images and videos
- Ensure responsive design (mobile-friendly)
- Make it visually impressive for client demo
- Use Vietnamese with full diacritics for all headings, labels, buttons, and descriptive text
- Do not mix English into the public-facing UI
```
